# Privacy Policy & User Instructions

## 1. General Information

This project, **Fifteen Puzzle Game (Pyatnashki)**, is a web-based educational application developed as part of an academic course project.  
The application is intended for learning, demonstration, and personal use only.

By using this application, you agree to the terms described in this document.

---

## 2. Purpose of the Application

The Fifteen Puzzle Game allows users to:
- Play the classic sliding puzzle game
- Choose different board sizes (3×3, 4×4, 5×5)
- Track game time and number of moves
- View previous game results

The application does not require user registration or authentication.

---

## 3. Data Collection and Storage

The application does **not** collect personal data such as:
- real names
- email addresses
- phone numbers
- IP addresses
- precise location data

### Local Storage
The following data may be stored locally in the user's browser:
- game settings (board size, difficulty)
- game results (time, moves)
- cookie consent choice

This data is stored using:
- `localStorage`
- browser cookies (for consent management)

All stored data remains on the user's device and is never transmitted to external servers.

---

## 4. Cookies Usage (GDPR Compliance)

This application uses cookies **only for functional purposes**, including:
- storing user consent preferences
- improving user experience

Upon first visit, users are presented with a cookie consent popup where they can:
- accept cookies
- decline non-essential cookies

Users can clear cookies at any time via their browser settings.
